#   Google 圖片爬蟲機器人 *

##  環境:

#### Node.js ver:1.1.7

##  檔案:

* google_img_bot.js --主要檔案

* package.json --套件管理工具

## 套件:
1. jquery: "^3.5.1"
2. jsdom": "^16.2.2"
3. moment": "^2.27.0"
4. nightmare": "^3.0.2"
5. base64-to-image": "^1.0.2
## 使用方法:
### 套件工具:
    npm i --save
**請務必注意目錄位置和package.json是否存在!**

### 執行:
    node google_img_bot.js
**注意套件使否有安裝!**
